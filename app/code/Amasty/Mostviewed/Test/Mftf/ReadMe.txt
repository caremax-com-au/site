Current version of tests is for Magento CE 2.3.3 with Amasty Automatic Related Products.
Release version consists of 12 smoke tests.  
The tests are divided into following groups:
- MostViewed (is used for running of all tests. E.g. vendor/bin/mftf run:group MostViewed -r)
- MostViewedSmoke (is used for running of all smoke tests)
